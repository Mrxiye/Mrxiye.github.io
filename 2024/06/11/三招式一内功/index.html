三招式一内功

招式一：矛盾构建（代入观众）

矛盾：人物非常想要做一件事情，但有东西阻碍着人物，导致人物做不了

矛盾的重要性：构成故事的核心部分，无矛盾就无故事，无矛盾的事情就是流水账

矛盾的核心作用：代入观众（观众会站在矛盾的一方） 。令观众设身处地的站在矛盾的其中一方，去体会人物在这个特定情境之下的感受， 进一步使观众理解人物，共情人物

矛盾构建的四个阶段：

第一阶段：矛盾的建立。

矛盾的双方（矛盾的主体）：我想干什么 + 谁在阻碍我 –> 外部因素

矛盾的动机： 我为什么想干 + 谁为什么阻碍我 –> 内部因素

动机构成内在动力，从而推动矛盾的发展，复杂的矛盾往往需要铺垫

构建矛盾的要求：

1.第一时间要快速构建矛盾，抛出悬念，快速构建矛盾只能是一目了然的以简单行为动机的矛盾，复杂的矛盾有一个复杂的行为动机，那么需要铺垫。

2.只有矛盾双方，没有动机，观众无法理解，更不会共情，就不能代入，被称为无效矛盾， 往往用动机去共情。

3.区分矛盾的属性：简单矛盾 和 复杂矛盾 成功的矛盾是成功的代入观众

第二阶段：矛盾的发展。（最有趣，最有意思、最重要的环节）

核心：变。 变的条件：灵活、灵动、松弛有度、出人预料的趋势才能做到矛盾的变化 主体和动机可以在发展中变化

第一阶段的四个问题也能变化，从而改变矛盾本身，矛盾得以发展 矛盾的激烈程度也在变化（缓和、紧张，不能一成不变）

矛盾发展的注意：

1.附加多重动机可以提升矛盾的层次感

2.生与死的矛盾的激烈程度是很高的，矛盾的激烈程度往往表现为一波三折

3.一波三折指：矛盾的缓和 和 紧张不同程度的交替三次 缓和的情况：对立到相互理解

4.区分 内在矛盾（如自己的懦弱阻碍自己杀人）和外在矛盾

5.矛盾的发展脉络 = 故事的发展脉络

矛盾是观众理解人物，感受人物的窗口

第三阶段：矛盾的激化

1.矛盾越是激化，想要合理化的手段就越少

2.矛盾的激化和化解是高度相关联：单纯的激化矛盾是件简单的事情，但是要化解矛盾，就 没有那么容易，所以考虑第三阶段的时候应该考虑第四阶段

3.完整的矛盾 = 矛盾的内在本质（主体和动机） + 矛盾的外在形式（冲突）

矛盾的内在本质是：

矛盾的双方（矛盾的主体）：我想干什么？ + 谁在阻碍我？

矛盾的动机： 我为什么想干？ + 谁为什么阻碍我？

矛盾的外在形式是：冲突 = 怎样去干？+怎样阻碍我

4.冲突的激烈程度 = 矛盾的激烈冲突

5.矛盾只有发展到穷尽所有的方法都不能调节才能被激化，否则只能缓解矛盾。

第四阶段：矛盾的化解

矛盾（主要矛盾）化解的方法：穷举法。

矛盾激化和化解的步骤：

第一步：先把矛盾激化到极致，

第二步：使用穷举法，依照剧情的逻辑列出所有剧情发展可能性

第三步：找到最优解：需要兼顾戏剧效果和合理性

第四步：在列出的可能性中找合理性

1.找出人物的行为动机。
做到复杂问题简单化+抽象问题具体化=说服力
为财？为名？为情？为义？为同理心？= 依据

2.做铺垫：便于观众接受这个理由。
往往在细节处铺垫（道具 + 画外音）
铺垫落脚点：人物的关系
铺垫内容：比如父爱->父亲穿儿子校服的细节等。

矛盾的层次：矛盾的主体和动机因素不止一个

矛盾的双方（矛盾的主体）：我想干什么 + 谁在阻碍我 –> 外部因素

矛盾的动机：我为什么想干（行为动机）+ 谁为什么阻碍我（阻碍动机）–> 内部因素

多重阻碍因素 和 多重动机：

作用：起到共情的作用

不同层面对应的不同动机 = 层次感（表面动机 和 深沉动机）， 把两个动机拆分到两个层面展示更加的清楚

同一层面的动机相同 而 不同阻碍 != 层次感

多重动机有一个主要的因素，做到主次分明

表层矛盾（具体）与里层矛盾（抽象）：

如表面是两个主体的矛盾，但里层则是更深层的矛盾（如个人权利 和 公权）。

有深度的片子具有里层和表层矛盾的联系

与矛盾层次感紧密相关：人物弧光

人物弧光：人物随剧情的发展而产生变化的轨迹

升华：坏人变好人

堕落：好人变坏人

人物弧光的产生：

外因（外）导致人物的原有动机变化或新增动机（内），导致行为变化（外） 难点在于，外因如何合理的改变人物的原有动机或新增动机。

方法是遵循最朴素的现实逻辑

注意人物弧光的外因不能变化的太快

矛盾的进阶：

1.矛盾的内在本质是：

矛盾的双方（矛盾的主体）：我想干什么？（想要一种结果） + 谁在阻碍我？ = 目的

矛盾的动机： 我为什么想干？ + 谁为什么阻碍我？=（外因产生）动机

矛盾的外在形式是：冲突 = 怎样去干？+怎样阻碍我？ = 行为（行为带有目的）

双方的行为是矛盾外化的体现，行为之间所产生的不同形式的对抗叫冲突， 双方的行为和目的决定了矛盾的内在本质，简单行为背后目的和动机是相同的

2.整个矛盾中的核心环节：人物动机

动机的主要作用：抽象的、是情感的落点，是观众共鸣的地方，而目的往往是具体的。

动机的作用：

决定行为的性质（善恶，高尚或卑劣） 为矛盾提供层次感：让观众认识角色 – 理解角色 – 共情角色 的桥梁 为双方的行为提供内在动力：矛盾的发展、激化、化解

3.痛感的产生：卑劣的动机 战胜 高尚的动机，寻常的动机 战胜 强大的动机

爽感的产生：高尚的动机 战胜 卑劣的动机，强大的动机 战胜 寻常的动机

所以人物的魅力由动机决定

招式二：悬念设置（留住观众）

悬念：观众想要知道的，但尚未知道的，会影响剧情走向的元素

悬念的作用：吸引广义观众不断地往下看

悬念的前置条件：悬念嗅觉和悬念意识

悬念嗅觉：精准掌控观众心理，从而知道观众想要知道的，然后设置剧情不让观众知道 也就是敏锐捕捉到故事原型中所有关键信息量的能力。

悬念意识：我们已经找到了观众想知道的，把他们藏起来

悬念大师：精准掌控观众心理，设置悬念，从而调动观众情绪、控制节奏、勾起好奇心 让观众去猜就让观众等一会，不猜就直接告诉观众 观众心理变化：完全猜到满足感/成就感，但猜到的多了就会无聊感，完全猜不到，理由不合理就会产生反感，合理则会产生落空感

设置悬念的本质：创作者在自己设定的剧情框架中，与观众展开一场心里博弈

设置悬念的工具：人物目的，动机，行为。

故事原型：一个按照时间顺序从头到尾完整发生的事件。

有了故事原型后，需要在六大问题中找到主要矛盾的发展脉络，然后对矛盾的四阶段进行改编，采用何种叙述方式，接着找到一个时间节点叙述故事结构（要便于设置悬念）

叙述结构：线索和时间（顺叙、倒叙、插叙、补叙）

如何高效的设置悬念：

1.关键信息量：缺一不可的信息

2.悬念围绕人物的目的，行为，动机的关键信息量来设置的，

往往是以人物的行为来设置悬念的，当动机和目的一目了然的时候，观众会对行为产生兴趣 对目的设置悬念，要么让观众猜测行为的目的，要么告诉观众行为目的让观众猜目的能否达成，对目的和动机设置悬念，只告诉行动，观众猜不到就会形成强悬念

3.隐藏动机或以动机设置为悬念，时间久了，会对矛盾构建造成影响， 因为动机在矛盾中是不可或缺的重要环节，

4.同一时间抛出多个悬念，散弹枪覆盖观众（考虑悬念的数量）

5.同一时间抛出一个很强（观众越想知道越强）的悬念（考虑悬念的强弱）

悬念的整体结构：

大悬念：贯穿始终的悬念（能不能达到目的呢），在主线最脆弱的时候不断

小悬念：在每一个情节被不断抛出和不断揭晓。必须与大悬念相关联 小悬念不断揭晓，不断推动剧情向大悬念的揭晓发展。

设置悬念的细节：

1.勾起观众的兴趣，隐藏关键信息量

2.分辨那些可以告诉观众让观众欣然接受，那些让观众去猜测

3.时而让观众产生满足感，防止过多产生无聊感，时而产生落空感，防止过多产生反感

悬念与矛盾的关系：

1.矛盾是整个影片的基本盘。

2.悬念是依附于矛盾构建进行的紧密相关，并且任何矛盾都自带悬念， 但是矛盾自带的悬念比较弱，则需要设置悬念来加强，称为矛盾悬念组合拳， 在带入观众的同时，就能起到吸引观众往下看

3.矛盾构建过程：简单矛盾在矛盾发展的时候复杂化，在每个矛盾的阶段都应该设置小悬念。

招式三：错误引导（杀死观众，反转）

错误引导的本质：通过设计情节，在恰当的时机，颠覆观众内心原本就有的，创作者传递给观众的认知，颠覆常理认知的效果 > 颠覆剧情认知

要求：

1.绝对不能让观众猜到-隐蔽措施：让观众猜不到结局

2.必须让观众相信-去除主观感受

高客观隐蔽性：被颠覆比较少的那种认知。如找到平平无奇的人或东西或双线来隐蔽

颠覆剧情认知：瞬发性伤害

颠覆常理性认知：延续性伤害

打击精度：跟主题是否相关联。

错误引导的要素：

1.隐蔽性 客观隐蔽性 隐蔽措施

2.合理性 令观众信服

3.打击精度

一内功：塑造人物

事归事，人归人，什么样的人才能干出什么样的事。

编事简单，编人难

塑造人物的方法：

编好事件

塑造角色：一个令观众信服，足以完成事件

确定事件->编排人物（什么样的人才能做出这样的事情，如何呈现）-> 令观众信服

塑造人物的意识：将编排事件转移到编排人物上，已完成时间事件

塑造人物的三个层面（自上为往下的基础）：

1.生动形象：外貌，语言，动作满足90%的普遍标签形象，10%突破刻板印象形成特色

2.立体全面：人物具有多面性，内心世界的多面性，作用提高真实感。 角色分为：主角（立体全面），配角（立体全面）， 功能性角色（可以扁平），群众角色（可以扁平）

3.有生命力：指当人物的行为动机与观众达成了一定程度的共情 以至于观众可以完全的感知角色的行为， 角色因此就有了生命力。(察觉不到创作者的身影)

塑造人物成功的标准：

1.人物是否令观众信服

2.人物是否有魅力

魅力1：给角色营造神秘感，让人物自带悬念

魅力2：角色做出我们想做却做不了的事情

观众对人物的三种心里状态：

1.完全相信角色的行为

2.一定程度上质疑角色的行为

3.对角色无感：

第一种：完全失败的角色，对一个层面的生动形象都相去甚远

第二种：毫无特征的脸谱化的角色

塑造人物的雷区：

1.人物只能做出他可以做出的行为，绝对不能强行令人物做出他无法做出的行为，要做出难以置信的行为时就要做铺垫（让观众很好的去接受人物的行为）

2.塑造人物的时候用力过猛，过犹不及 塑造人物悲伤用反衬，乐景衬托哀情。

铺垫：

1.铺什么：参考你想让观众接受什么

2.往哪里铺：往细节铺（因为不容易被人发现，还可以深入人心）

3.铺多少：（能谱多少就铺多少）
